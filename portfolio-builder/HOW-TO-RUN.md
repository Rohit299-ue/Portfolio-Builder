# How to Run Portfolio Builder

## Quick Start

1. **Start the application**:
   ```
   Double-click: START-APP.bat
   ```

2. **Wait for**:
   ```
   "Started PortfolioApplication"
   ```

3. **Open browser**:
   ```
   http://localhost:8080
   ```

## First Time Setup

1. Click "Get Started Free"
2. Create your account
3. Login to dashboard

## Using the Dashboard

### Upload Photos (2 required)
- **Hero Photo**: Appears at top of portfolio
- **About Photo**: Appears in about section with gradient frame

### Fill Profile Information
- **Hero Section Bio**: Short introduction (2-3 sentences)
- **About Section Text**: Detailed description (longer)
- **Skills**: Comma-separated list
- **Experience & Education**: Your background

### Add Projects
- Title, description, technologies
- GitHub and live demo URLs

### View Portfolio
- Click "View Portfolio" button
- Share the URL with employers

### Download Portfolio
- HTML file (standalone)
- PDF document
- ZIP package (complete)

## Troubleshooting

### Port 8080 already in use
```
Right-click KILL-PORT-8080.ps1 ‚Üí Run with PowerShell
Then run START-APP.bat
```

### Application won't start
```
Check Java is installed: java -version
Make sure you're in portfolio-builder folder
```

### Photos not uploading
```
Check file size (under 10MB)
Use JPG or PNG format
Restart application
```
